A repository to organize and track job applications effectively.

## Repository Structure

```

├── templates/           # Reusable document templates
├── documents/          # Core documents (CV, references)
└── applications/       # All job applications
    ├── to-apply/      # Interesting positions to apply to

    └── applied/       # Completed applications
```

## Usage Guidelines

1. **Adding New Opportunities**
   - Create new folder in `to-apply/`
   - Copy job description immediately
   - Fill out basic tracker.md

2. **Working on Applications**
   - Move folder to `applying/`
   - Use template from `templates/`
   - Track progress in tracker.md

3. **After Submission**
   - Move folder to `applied/`
   - Keep copies of all submitted documents
   - Log all communications
   - Update status in this README

4. **Naming Conventions**
   - Folders: `company-name-YYYY`
   - Files: lowercase with hyphens
   - Dates: YYYY-MM-DD format

5. **Document Updates**
   - Keep this README current
   - Update status table regularly
   - Archive completed applications
